# Plonk's Blog

个人博客网站 —— 记录技术、分享生活、传递思考。

## 功能

- **首页** — 展示最新文章和资源
- **关于页面** — 个人信息展示，GitHub 链接
- **联系页面** — 通过邮件发送建议和反馈

## 技术栈

- HTML5 + CSS3 + JavaScript（纯静态）
- 响应式设计，适配 PC 和移动端
- 由 GitHub Pages 托管部署

## 如何部署

1. 将本项目 push 到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为部署源
